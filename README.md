# Cyber Shield 

Cyber Shield is a full-stack web application that detects suspicious or potentially malicious files using AI.

## 🚀 Features
- User authentication (login/signup)
- File and image analysis
- AI-based classification (Safe / Suspicious / Document / NSFW)
- Real-time results

## 🛠️ Tech Stack
- React (Frontend)
- Node.js (Backend)
- MongoDB (Database)
- Google Gemini API
- 
## 🔄 How It Works
Frontend (React) → Backend (Node.js) → Gemini API → MongoDB  

## ⚙️ How to Run

### Backend
cd backend  
npm install  
npm start  

### Frontend
cd frontend  
npm install  
npm run dev  

Open: http://localhost:3000
